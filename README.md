## 🔍 What the challenge does
- Fetches the Wikipedia page on Data Science
- Cleans and extracts the article text
- Identifies the most important keywords and phrases
- Visualises results as a bar chart and word cloud

## 🛠️ Libraries Used
- `BeautifulSoup` — web scraping
- `rake-nltk` — keyword extraction
- `wordcloud` — visualisation
- `matplotlib` — plotting
